# BDD Automation


## Getting Started

pip install -r requirements.txt

### Prerequisities

```
Python version >= 3.7
```

```
Supported Browsers(Chrome and ChromeHeadless)
```

## Running the tests

```
To run scripts for uitests: behave features/modules --tags=wip
```

* parse >= 1.6.3
* parse_type >= 0.3.4
* selenium >= 2.53.1
* argparse >= 1.4.0
* PyHamcrest >= 1.8
* allure-behave
